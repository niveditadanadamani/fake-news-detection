🚀 **AI-Based Fake News Detection System**

🔍 Developed an intelligent web application to detect and classify news as **Real, Fake, or Uncertain** using Machine Learning and NLP.

🧠 **Key Features:**
✨ Real-time news analysis
📊 Confidence score for predictions
💡 Explanation of results for transparency
🎨 Clean and responsive UI

⚙️ **Tech Stack:**
💻 Python (Flask)
🌐 HTML, CSS, JavaScript
📈 Scikit-learn (Logistic Regression)
🔤 TF-IDF Vectorization

🎯 Built to address misinformation and help users verify news content effectively.
